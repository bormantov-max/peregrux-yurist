# Peregrux Yurist Landing

Одностраничный лендинг для юридической помощи перевозчикам и владельцам грузового транспорта по вопросам оспаривания штрафов за перегруз по массе и осям.

## Цель проекта

Создать конверсионный лендинг, который помогает пользователю быстро понять:

- что сайт посвящён именно штрафам за перегруз;
- с какими типами постановлений можно обратиться;
- как проходит первичный анализ;
- как связаться со специалистом;
- какие документы нужно отправить для оценки перспектив обжалования.

Главная цель сайта — получить заявку на первичный анализ постановления.

## Ниша

Юридическая помощь при оспаривании штрафов за:

- превышение общей массы;
- превышение осевой нагрузки;
- автоматические весовые рамки АСВГК / АПВГК;
- постановления Ространснадзора;
- штрафы с весового контроля;
- спорные ситуации по документам перевозки.

География: вся Россия.  
Формат работы: дистанционно.

## Целевая аудитория

- владельцы грузового транспорта;
- ИП-перевозчики;
- транспортные компании;
- логистические компании;
- собственники ТС, получившие постановление о штрафе.

## Основное целевое действие
